Detailed Description

Security 2 (S2) Commmand Class handler and control functions.

This module takes care of the S2 Command Class support and control.

It uses a subscription to ATTRIBUTE_COMMAND_CLASS_SECURITY_2_VERSION in the Attribute Store to find out about the nodes that support Security 2.
