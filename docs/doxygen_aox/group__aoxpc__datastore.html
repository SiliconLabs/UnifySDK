◆ aoxpc_datastore_version

enum aoxpc_datastore_version

Versioning of the AoXPC datastore. Each datastore version identifies the unique schema of the SQL Schema / set of keys/values that is saved using the datastore. The version number should be incremented when new schema is introduced in datastore which is no more compatible with current or old schemas.

Enumerator
DATASTORE_VERSION_V1 

Initial AoXPC datastore revision at first release.

DATASTORE_VERSION_LAST 

Keep this value after the latest version to detect what is the latest.
