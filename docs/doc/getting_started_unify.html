Getting Started with Unify

Introduction

Unify SDK is a software framework that simplifies the developer experience, removing difficult parts of network control and network management as it relates to gateway and hub development in IoT applications. It is extended to support Z-Wave protocol using Z-Wave Protocol Controller. Matter Bridge serves as a reference IoT service, providing the capability to connect Matter to protocols that do not natively run .

Prerequisites

  • An RPi4 preinstalled with Raspberry Pi OS Linux Version 12, Bookworm 64-bit

The following are the ways to get started with Unify…

  1. Use Portable Runtime for quick demo with Unify IoT gateway running on Windows, MacOS or Linux desktop.

  1. Use Unify on Raspberry Pi for complete hands-on of Unify IoT gateway.
